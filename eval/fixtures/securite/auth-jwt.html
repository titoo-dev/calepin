Le refresh token est à usage unique, remplacé à chaque utilisation. Un refresh token volé et réutilisé plusieurs fois doit pouvoir être détecté. La réutilisation d'un refresh token déjà consommé révoque toute la famille de tokens. Ne jamais renvoyer le même refresh token deux fois. src/auth/token-service.ts

Cette détection de réutilisation attrape le vol de token même sans savoir comment il a fuité.
