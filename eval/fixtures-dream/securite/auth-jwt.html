Le refresh token est a usage unique, remplace a chaque utilisation. Un refresh token vole et reutilise plusieurs fois doit pouvoir etre detecte. La reutilisation d un refresh token deja consomme revoque toute la famille de tokens. Ne jamais renvoyer le meme refresh token deux fois. src/auth/token-service.ts

Cette detection de reutilisation attrape le vol de token meme sans savoir comment il a fuite.
