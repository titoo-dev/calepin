# Code de conduite

Ce projet adopte le [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) (texte canonique, [version française](https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/)).

En résumé : environnement ouvert et accueillant pour tous ; empathie et bienveillance ; les désaccords se règlent sur les idées, jamais sur les personnes ; pas de harcèlement, d'attaques personnelles ni de contenu déplacé.

Signalement d'un comportement inacceptable : en privé aux mainteneurs via
[GitHub](https://github.com/titoo-dev/calepin/security/advisories/new) ou tout canal privé listé sur le profil du mainteneur. Les signalements sont traités avec discrétion.
